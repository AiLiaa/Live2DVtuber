## 使用

### 运行

其中`py`和`unitylive2D`文件夹是项目源码

`Live2D_V2.0`和`py_V2.3`是已打包文件

**运行：执行 `VtuberLive2D.exe`，再执行 `py_V2.3\dist\main\main.exe`**

**关闭：关闭`VtuberLive2D`窗口即可**

`python`和`unity`进程通信实现：建立TCP连接，将`python`数据传输到`unity`
